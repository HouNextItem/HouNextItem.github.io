AI PM Demo Prototype

心镜方AI

一个面向大学生焦虑识别与分层干预的多智能体心理支持原型。用最小可运行页面，把“测评 - 画像 - 干预 - 督导”闭环直观展示出来。

多 Agent 协作 中医情志 + CBT 风险分层 可视化画像 高风险转介

展示定位

更像一个完整产品，而不只是一个聊天机器人。

面试看点

把产品结构、用户旅程、风险兜底和 AI 逻辑讲清楚。

当前阶段

前端交互原型，可直接打开演示，无需安装依赖。

1. 测评 Agent

采集用户在学业、社交、睡眠和数字内耗上的即时状态。

输入 4 个维度的自评强度，形成基础风险信号。

2. 画像 Agent

把分数转成“情境 - 情志 - 时代特征”的可视化画像。

让结果不只是一串分数，而是更好解释的结构化画像。

3. 干预 Agent

按风险等级提供中医情志调摄、呼吸练习和认知建议。

强调轻量、日常、可执行，而不是医疗诊断。

4. 督导 Agent

当风险过高时触发人工求助与校园资源转介提示。

体现安全边界和产品伦理，是展示里很重要的一环。

快速测评

拖动滑块，模拟一次最小版焦虑状态评估。
